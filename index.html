Git Basic Command List

Serial Git Command Description Tags
1 git init Initialize git repository in a folder init,start git,open git
2 git status Information about changed/deleted files status,condition,changes,modify
2 git config username.name="Jhon" Config git user for the current git git, config, user,name
2 git config username.email="jhon@gmail.com" Config git email for the current git git, config, user,email
2 git config --global username.name="Jhon" Config git user for the current git globally in a machine setup,config, globally, user,name
2 git config --global username.email="jd@gmail.com" Config git email for the current git globally in a machine setup,config, globally, user,email
2 git remote add origin https://github.com/arif98741/gitcom Config remote connection for uploading files and data to the repository(such as github, gitlab, bitbucket, sourceforge) setup,origin,remote,server,repo
2 git -v Current installed git version git,version,show
3 git add file.txt Add specific file for commit add, fixed, specific, custom file
4 git add   . Add all changed file for commit add, all, every file, changed
5 git commit -m "commit message" Commit Changes with messages and information commit, change, -m
5 git branch See all branch list branch, list, local
6 git branch branch_name Create new branch locally branch, change, make, create
7 git checkout -b branch_name Create new branch and move to there create, move, branch, checkout
6 git checkout -b branch_name
git push origin branch_name 		Create new branch locally and push it to origin branch, change, make, create
8 git checkout dev Migrate from any branch to dev create, move, branch, checkout
9 git pull update local branch by fetching data from remote origin pull, update,remote ,origin
9 git tag Show all local tag list tag, list, all, tags
9 git ls-remote --tags origin Show all remote tag list tag, list,remote, origin, all, tags
9 git tag tag_name Create a new tag locally tag, create, new, local
9 git tag tag_name -a

9 git tag -d tag_name Delete local tag tag,delete,clear, local
9 git push origin :tagname
git push --delete origin tagname		 Delete remote tag tag, delete, push, origin
9 git fetch --tags Fetch and update local tags from remote origin tag,fetch,remote,origin
